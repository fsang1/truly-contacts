### Install server

`npm install -g json-server`

Note if you are on a mac you need to run `sudo npm install -g json-server`

### Start server
`json-server --watch db.json` from the project `root` directory

### `npm start`

Once the server has started open a new terminal tab and run start the app by running `npm start`. Say yes to running on a different port than 300 since the server is already runnong on port 3000. Once the app starts you will be given a link to open in the browser.
